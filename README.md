🏦 BancoManeiro - Simulação de Terminal Bancário
Este projeto foi desenvolvido como parte de um desafio técnico para praticar os fundamentos da linguagem Java. O objetivo é simular o fluxo de criação de uma conta bancária via terminal, interagindo com o usuário para capturar dados e exibindo uma mensagem de confirmação formatada.

🚀 Funcionalidades
Captura de Dados: Recebe nome, agência, número da conta e saldo via console.

Validação de Entrada: Possui um sistema de validação que impede que o usuário avance com respostas inválidas no menu final.

Loop de Reinicialização: Permite que o usuário crie múltiplas contas em sequência sem precisar reiniciar o programa manualmente.

🧠 Conceitos de Java Treinados
Neste projeto, apliquei e consolidei os seguintes pontos da linguagem:

Entrada e Saída de Dados: Uso da classe Scanner para capturar diferentes tipos de dados (String e double) e System.out.println para interação com o usuário.

Manipulação de Strings: * Concatenação de variáveis para exibição de mensagens.

Uso do método .equalsIgnoreCase() por opção minha para comparar textos ignorando a diferença entre maiúsculas e minúsculas.

Estruturas de Controle e Repetição:

while: Implementação de um loop de validação com operadores lógicos de negação (!) e conjunção (&&).

if-else: Tomada de decisão baseada na resposta do usuário.

Recursividade de Método: Utilização da chamada do método main(args) dentro de si mesmo para reiniciar o fluxo do programa (conceito inicial de recursão).

Tipagem de Dados: Trabalho com tipos primitivos (double) e objetos (String), entendendo a importância da precisão decimal para saldos bancários.
